# Завдання 15. Перевірка доступності застосунку

1. Спочатку виконав `python manage.py migrate`, щоб база даних створила потрібні таблиці.
2. Запустив сервер командою:
   ```bash
   python manage.py runserver
   ```
3. У браузері (та через `curl`) відкрив `http://127.0.0.1:8000/` і побачив повідомлення
   “Вітаю в магазині! Це тестове повідомлення.” з view `shop.views.index`.
4. Сторінка `http://127.0.0.1:8000/admin/` тепер показує форму входу, бо міграції вже застосовано.

> Отже, застосунок доступний, і обидві головні сторінки (корінь та адмінка) відповідають очікуванню.
