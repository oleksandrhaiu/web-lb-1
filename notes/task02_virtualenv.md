# Завдання 2. Створення віртуального середовища

1. У корені проєкту виконав команду:
   ```bash
   python3 -m venv venv
   ```
2. Переконався, що з'явилася папка `venv/` з підкаталогами `bin/`, `lib/`, `include/`.
3. Активував середовище (`source venv/bin/activate`) і далі працюю вже в ньому.

> Це завдання не дає нових треканих файлів, бо саму теку `venv/` ми ігноруємо через `.gitignore`, але коміт фіксує факт виконання кроку.
