# Завдання 6. Перевірка, що Django стартує

1. Активував віртуальне середовище та запустив команду:
   ```bash
   python manage.py runserver
   ```
2. У браузері відкрив `http://127.0.0.1:8000/` і побачив стандартну “Congratulations!” сторінку Django.
3. Після цього зупинив сервер комбінацією `Ctrl+C`.

> На цьому етапі ще не виконував міграцій, тому адмін-панель не працює (це наступне завдання).
